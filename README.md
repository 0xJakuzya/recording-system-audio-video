# REC SYSTEM — Запись видео/аудио с метками

Десктопное приложение на Python/Tkinter для синхронной записи аудио и видео с расстановкой временных меток. Удобно для лабораторных экспериментов, интервью, тестирования usability.

## Возможности

- Запись аудио (WAV, 44100 Hz, mono) и видео (AVI/MJPEG) одновременно
- Расстановка временных меток (начало / конец сегмента) горячей клавишей
- Живой превью с камеры в интерфейсе
- Сохранение меток в JSON с временем в секундах от старта
- Graceful degradation — запись продолжается без недоступных устройств

## Установка

```bash
git clone https://github.com/<ваш-логин>/metka-vido-audio.git
cd metka-vido-audio
python -m venv .venv
.venv\Scripts\activate     
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Управление

| Действие | Способ |
|---|---|
| Начать / остановить запись | Кнопка **НАЧАТЬ ЗАПИСЬ** |
| Поставить метку (начало/конец сегмента) | Кнопка **МЕТКА** или **Пробел** |
| Полноэкранный режим | `F11` |
| Выйти из полноэкранного режима | `Escape` |

Первый пробел — начало сегмента (зелёный индикатор).  
Второй пробел — конец сегмента (показывается длительность).

## Структура выходных файлов

Каждая сессия сохраняется в `data/session_N_YYYYMMDD_HHMMSS/`:

| Файл | Описание |
|---|---|
| `audio_YYYYMMDD_HHMMSS.wav` | Аудио (44100 Hz, mono, 16-bit PCM) |
| `video_YYYYMMDD_HHMMSS.avi` | Видео с камеры (MJPEG, 20 fps) |
| `markers_YYYYMMDD_HHMMSS.json` | Временны́е метки |

Формат `markers.json`:

```json
{
  "markers": [
    { "type": "start", "time": 12.345, "segment": 1 },
    { "type": "end",   "time": 18.710, "segment": 1 }
  ]
}
```

`time` — секунды от начала записи.

## Требования

- Python 3.9+
- Микрофон
- Веб-камера (опционально — без камеры аудио и метки всё равно пишутся)
- tkinter (входит в стандартный Python на Windows/macOS)
